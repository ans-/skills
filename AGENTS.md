# AGENTS.md

## 项目描述

这是一个技能（Skills）集合项目，用于为大模型提供各种实用技能。每个技能都是一个独立的子目录，包含 `SKILL.md` 文件定义技能的功能和使用方法。

## 项目结构

```
skills/
├── [skill-name]/
│   ├── SKILL.md          # 技能定义文件
│   └── references/        # 参考文档（可选）
└── README.md
```

## 可用技能

| 技能名称  | 描述                                           |
| --------- | ---------------------------------------------- |
| storybook | 生成个性化儿童绘本，支持多种艺术风格和年龄适配 |

## 添加新技能

如需添加新技能，请按以下步骤操作：

1. 在 `skills/` 目录下创建新技能文件夹
2. 创建 `SKILL.md` 文件，定义技能功能
3. 使用 YAML front matter 包含 `name` 和 `description` 字段

### SKILL.md 格式要求

```yaml
---
name: skill-name
description: 技能描述（用于技能发现）
---

# 技能名称

## 功能说明

## 使用方法

## 参数说明

| 参数 | 说明 | 默认值 |
|-----|------|-------|
|      |       |       |
```

## 约定规范

- 所有技能文档使用英文文件名
- 技能描述使用英文（用于大模型技能发现）
- 内部文档注释可使用中文
- 使用 YAML front matter 定义元数据
- 参考文档放在 `references/` 子目录

## 常用命令

- `ls skills/` - 列出所有可用技能
- 查看具体技能的 `SKILL.md` 文件了解使用方法
