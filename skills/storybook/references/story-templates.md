# Story Templates

Story structure templates for different page counts and themes.

## 6-Page Template (Default)

| Page  | Purpose       | Content                                   |
| ----- | ------------- | ----------------------------------------- |
| Cover | Title page    | Story title + main character illustration |
| 1     | Opening       | Introduce protagonist and setting         |
| 2     | Development   | Establish the situation or problem        |
| 3     | Rising Action | Character takes action                    |
| 4     | Climax        | Peak moment of the story                  |
| 5     | Resolution    | Happy ending with lesson learned          |

## 4-Page Template (Minimum)

| Page  | Purpose              |
| ----- | -------------------- |
| Cover | Title + illustration |
| 1     | Opening + problem    |
| 2     | Action + climax      |
| 3     | Resolution           |

## 10-Page Template (Maximum)

| Page  | Purpose                        |
| ----- | ------------------------------ |
| Cover | Title + illustration           |
| 1-2   | Opening: world-building        |
| 3-4   | Development: deepen characters |
| 5-6   | Rising Action: build tension   |
| 7-8   | Climax: peak moments           |
| 9-10  | Resolution: conclusion         |

## Theme Examples

### Educational Story

- **Goal**: Teach a concept (science, nature, social skills)
- **Structure**: Curious character → discovers/learns → shares knowledge
- **Example**: "Why do leaves change color?"

### Adventure Story

- **Goal**: Exciting journey with challenges
- **Structure**: Call to adventure → obstacles → triumph
- **Example**: "The little boat's ocean voyage"

### Emotional Story

- **Goal**: Help process feelings
- **Structure**: Character faces emotion → understands it → grows
- **Example**: "When I feel scared"
