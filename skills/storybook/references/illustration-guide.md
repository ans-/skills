# Illustration Guide

Detailed guidelines for creating consistent, high-quality storybook illustrations.

## Illustration Prompt Template

```
[Art style] illustration for children's book. [Scene description].
Characters: [Character details for consistency].
Style: [Specific style notes]. Warm, friendly colors, child-appropriate.
No text, no words, no labels. (For cover: include title and subtitle only, no author info)
```

## Cover Illustration Rules

- **Include**: Title and subtitle text only
- **Exclude**: Author names, publisher info, watermarks, or any other text
- Add to prompt: "No author name, no publisher info, no watermarks, title only"

## Consistency Checklist

For each page, maintain consistent:

- **Character descriptions**: Same appearance throughout (hair, clothing, features)
- **Art style**: Match user's chosen style exactly
- **Color palette**: Child-friendly, cohesive across all pages
- **Composition**: Clear focal point, simple backgrounds

## Language in Illustrations

If the storybook has a specified language, ensure any text appearing in illustrations (signs, books, banners, etc.) matches that language:

| Language | Example Text   |
| -------- | -------------- |
| zh-Hans  | 中文标题       |
| en       | English Title  |
| ja       | 日本語タイトル |

## Art Style Reference

| Style         | Description                                         | Prompt Keywords                                                                   |
| ------------- | --------------------------------------------------- | --------------------------------------------------------------------------------- |
| watercolor    | Soft, flowing colors with visible brushstrokes      | soft watercolor, gentle pastel colors, visible brushstrokes, dreamy atmosphere    |
| cartoon       | Bold outlines, bright colors, expressive characters | bold outlines, bright colors, expressive, Disney-inspired                         |
| pixel-art     | Retro 8-bit/16-bit style                            | 16-bit pixel art, retro video game aesthetic, vibrant colors, clear pixel details |
| claymation    | 3D clay-like textured characters                    | clay-textured, 3D rendered, stop-motion style                                     |
| comic         | Panel-style with dynamic poses                      | comic book style, dynamic poses, action lines                                     |
| coloring-book | Black outlines only, for printing and coloring      | black outlines only, no fill, coloring page, clean lines                          |

## Prompt Examples by Style

### Cartoon

```
Cartoon illustration for children's picture book. A cheerful brown mouse wearing
a tiny red cape stands bravely on a mushroom, looking at a starry night sky.
Warm colors, soft lighting, Disney-inspired style. Simple forest background.
```

### Watercolor

```
Soft watercolor illustration for children's book. A little girl with pigtails
holding hands with her grandmother in a flower garden. Gentle pastel colors,
visible brushstrokes, dreamy atmosphere. White background edges fading softly.
```

### Pixel Art

```
16-bit pixel art illustration. A brave knight facing a friendly dragon in a
colorful castle courtyard. Retro video game aesthetic, vibrant colors,
clear pixel details. Blue sky with fluffy cloud sprites.
```
